# Changelog

## 0.9.0 - 2026-05-15

### Added

- 新增 `赛事查询` 命令，支持按日期、赛区、战队、场次查询 RoboMaster 赛程与比分，并支持自然语言问题解析。
- 赛事数据源新增 `auto` / `schedule` / `official` 模式：默认优先使用 `schedule.scutbot.cn`，失败后回退官方 `live_json`。
- 赛事单场详情支持展示小程序投票比例和 B 站回放；历史交手查询支持 schedule API 数据源。
- 新增赛事查询 LLM 理解配置，可将“今天有哪些比赛”“南部赛区第12场是谁打谁”等问题解析为结构化查询参数。
- 新增赛事信息图片实验功能，可将真实赛事查询结果渲染为 RM Schedule 风格表格或流程图图片；Playwright 或 Chromium 不可用时自动退回文字。
- 新增赛事查询长结果合并转发，QQ/OneBot 中赛程较长时可拆入合并转发节点，减少刷屏。
- 新增公告、赛事、论坛开源的独立订阅命令与推送会话配置：`RM订阅公告`、`RM订阅赛事`、`RM订阅开源` 及对应取消订阅命令。
- 新增公告、赛事、论坛开源的飞书卡片通知配置，卡片不可用时自动降级为普通文本。
- 新增外部 Webhook 推送配置，可将公告和赛事事件 POST 到自建服务。
- LLM Tools 新增赛事查询 Tool，并补充公告、赛事、论坛开源独立订阅相关 Tool。

### Changed

- 赛事监控复用新的赛事数据层，支持赛区白名单、独立推送会话和飞书卡片状态展示。
- 通知订阅从旧的通用 `RM订阅通知` 迁移为公告、赛事、论坛开源三套独立订阅列表，配置页目标与命令订阅目标会合并去重。
- 配置 schema 重组为更清晰的分组，补充赛事数据源、赛事查询、赛事信息图片、长消息发送、Webhook 和飞书卡片等配置说明。
- README 重写赛事查询、赛事推送、通知订阅和 Playwright 安装教程，并精简 LLM Tools、规则手册配置和推送出口说明。
- 插件版本号更新为 `0.9.0`。

### Fixed

- 修复飞书混合图文消息可能丢失文字的问题，规则手册回复可按配置拆分文字和图片发送。
- 优化论坛监控和导入流程的锁粒度与状态记录，减少长任务阻塞其他论坛查询。
- 优化手册下载、截图缓存和检索索引的存储路径处理，统一使用插件数据目录。

## 0.8.0 - 2026-05-14

### Added

- 新增 AstrBot LLM Tools 支持，通过 `context.add_llm_tools(...)` 注册规则手册、论坛开源、监控状态和管理员操作入口。
- `rm_manual_search` 复用原规则手册命令的图文回复流程，可按配置发送文本、截图、合并转发或完整图文结果。
- 规则手册索引为空并触发首次自动构建时，会先提示用户正在构建索引。

### Changed

- 论坛 JSONL 导入改为逐行读取，并将新增文章的 LLM 摘要生成移到服务锁外，减少导入期间对其他论坛功能的阻塞。
- 论坛开源查询的 LLM 选择失败时，会记录截断后的响应片段，便于排查模型未返回合法 JSON 的情况。
- README 增加 LLM Tools 用法说明，并更新相关 TODO 状态。

### Fixed

- 修复赛事小局显示与官方接口状态偏移的问题：比赛开始时显示上一状态 `- / N`，单局结束后显示已结束小局数，避免满局后出现异常的 `- / N`。
- 修复规则手册自动构建索引的并发边界：多个会话同时首次查询时，锁内会二次检查全局索引状态，避免重复重建。

## 0.7.4 - 2026-05-06

### Added

- 新增独立的公告、赛事、论坛和通知命令/服务模块，拆分原集中式监控实现。
- 新增后台任务管理模块，统一管理公告、赛事和论坛监控任务的启动、停止与状态输出。
- 新增 `RM监控状态` 等命令处理封装，便于查看后台任务和订阅状态。

### Changed

- 重构插件主入口，将规则手册、论坛开源、通知订阅、公告监控和赛事监控拆分到独立 handler/service。
- 将监控状态存储从 `monitors` 迁移到 `core.state`，公告和赛事事件模型按业务模块重新组织。
- README 重写快速开始、命令列表、功能配置、论坛开源资料、公告通知、赛事通知和贡献说明。

### Fixed

- 优化后台监控任务启动时机和幂等性，减少重复启动或状态不清晰的问题。

## 0.7.3 - 2026-04-30

### Fixed

- 修复 `Main.__init__` 签名与框架不一致的潜在问题：`config` 参数改为可选，避免框架未注入时实例化报错。
- 修复 `search_manual` 中"规则手册帮助"分支未调用 `_stop_event`，可能导致与 `manual_help_command` 重复响应。

### Changed

- 将 `_send_forum_progress` 中的 f-string 日志改为延迟格式化，减少无效字符串拼接。
- 提取 `normalize_text` / `tokenize` 到共享模块 `core/text_utils.py`，消除 `forum/search_index.py` 与 `manual/search_engine.py` 间的重复代码。

## 0.7.2 - 2026-04-30

### Fixed
- 修复论坛列表解析器遇到头像、封面等 `<img>` 标签后无法结束文章卡片解析，导致 browser 模式抓到页面却显示列表 0 篇的问题。
